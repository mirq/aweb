struct Plugininfo

Purpose

The struct Plugininfo is used for holding the results of the Initplugin() function of a plugin module.

Structure contents

struct Plugininfo
{  ULONG sourcedriver;
   ULONG copydriver;
};

Usage

Field: sourcedriver

Class ID (or object type) for the plugin sourcedriver class.

Field: copydriver

Class ID (or object type) for the plugin copydriver class, or zero if the plugin module doesn't implement a copydriver.
