struct Arect

Purpose

The struct Arect contains the coordinates of a rectangle.

Structure contents

struct Arect               /* a rectangle */
{  LONG minx,miny;
   LONG maxx,maxy;
};

Usage

Field: minx, miny

These are the coordinates (relative to some origin) of the upper left corner of the rectangle.

Field: maxx, maxy

These are the coordinates (relative to some origin) of the bottom right corner of the rectangle.
