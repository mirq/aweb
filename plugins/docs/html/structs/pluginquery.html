struct Pluginquery

Purpose

The struct Pluginquery is used for holding the results of the Queryplugin() function of a plugin module.

Structure contents

struct Pluginquery
{  long structsize;        /* Set to struct size by caller */
   BOOL command;           /* Set to TRUE if plugin supports Commandplugin() */
   BOOL filter;            /* Set to TRUE if plugin supports Filterplugin() */
};

Usage

Field: structsize (in)

Size of this structure, in bytes. The exact size of this structure may vary with each version of AWeb.

Do not touch parts of this structure beyond the size indicated by this field. That is, support earlier versions of AWeb.

Do not touch parts of the structure beyond the size the plugin expects this structure to be. That is, don't touch parts of the structure you don't understand.

Field: command (out)

Should be set to nonzero by the plugin if the plugin supports the Commandplugin() function.

Field: filter (out)

Should be set to nonzero by the plugin if the plugin supports the Filterplugin() function.
