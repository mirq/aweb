Queryplugin()

Purpose

Interrogation of the plugin module for the additional features it supports.

Synopsis

   Queryplugin(pluginquery)
               A0

   void Queryplugin(struct Pluginquery *)

Inputs

Parameter Description
pluginquery Pointer to a struct Pluginquery.

Implementation

This function is called at least once before any additional feature of the plugin is used.

This function should set the boolean fields in the supplied Pluginquery structure for each additional feature that is supported by the plugin module.

The structsize field of this structure is set to its size before this function is called. Do not modify any fields beyond this size.

Compatibility

In pre-2.0 versions of the API this function was not defined. In order to gain backward compatibility, this function and all additional features are assumed to be absent if the plugins lib_NegSize is less than 36.

Example

Both examples of normal and filter type plugins contain an example implementation of the Queryplugin() function.
