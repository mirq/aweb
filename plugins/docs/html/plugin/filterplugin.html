Filterplugin()

Purpose

This function is used to filter data from data blocks read by AWeb, and write back data to be processed.

Synopsis

   Filterplugin(pluginfilter)
                A0

   void Filterplugin(struct Pluginfilter *)

Inputs

Parameter Description
pluginfilter Pointer to a struct Pluginfilter.

Implementation

This function is optional. It is only supposed to be present after the Queryplugin() function indicated its presence.

If supported, the function should examine any data in the data block pointed to by the struct Pluginfilter.

Optionally data can be written back to AWeb to be processed, with the Writefilter() function. This data may be the same as the input, modified data, or totally new data. It is allowed to write back no data at all, or only after eof has been reached.

Example

The example filter type plugin contains an example implementation of the Filterplugin() function.
