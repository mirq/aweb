Commandplugin()

Purpose

This function is used to pass a plugin-specific textual command to the plugin after a PLUGIN ARexx command was received by AWeb.

Synopsis

   Commandplugin(plugincommand)
                 A0

   void Commandplugin(struct Plugincommand *)

Inputs

Parameter Description
plugincommand Pointer to a struct Plugincommand.

Implementation

This function is optional. It is only supposed to be present after the Queryplugin() function indicated its presence.

If supported, the function should parse and execute the command passed in the struct Plugincommand. Optionally it can return a textual result in this structure. In case of failure a numeric result code should be passed in this structure.

Example

The example filter type plugin contains an example implementation of the Commandplugin() function.
