Setfiltertype()

Purpose

Set a new content type from within a filter type plugin. (V2.0)

Synopsis

   Setfiltertype(handle, type)
                 A0      A1

   void Setfiltertype(void *, char *)

Function

This function sets a new MIME content type from within a filter type plugin.

This function can only be called from within the plugins Filterplugin() function. It should be called before any data is written by the Writefilter() function, or else it will have no effect.

Inputs

Parameter Description
handle The handle contained in the Pluginfilter structure.
type A null-terminated string holding the new MIME content type (e.g. "text/html").

Currently only text types are supported (text/html, text/plain, etc).

Results

Subsequent data written by the Writefilter() function will be interpreted as being of this type.
