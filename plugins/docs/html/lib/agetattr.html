Agetattr()

Purpose

Perform the AOM_GET method on an object for one attribute.

Synopsis

   result = Agetattr(object, attributeID)
   D0                A0      D0

   ULONG Agetattr(struct Aobject *, ULONG)

Function

Object oriented support function that invokes the AOM_GET method on the specified object, to get the current value of one of the object's attributes. To get the value of more than one attribute, the AgetattrsA() function is more efficient.

Inputs

Parameter Description
object Pointer to object to invoke the method on
attributeID ID of the attribute to obtain

Results

The result of this function is the current value of the attribute.
