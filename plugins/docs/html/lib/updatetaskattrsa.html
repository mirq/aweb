UpdatetaskattrsA()

Purpose

Send an AOM_UPDATE message from a subtask to the main task.

Synopsis

   result = UpdatetaskattrsA(taglist)
   D0                        A0

   ULONG UpdatetaskattrsA(struct TagItem *)

   result = Updatetaskattrs(tag1, ...)

   ULONG Updatetaskattrs(ULONG, ...)

Function

AOTP_TASK support function. It builds an AOM_UPDATE message using the supplied taglist. Then it sends this message to the main task, invoking the target object's AOM_UPDATE method.

If you include the AOTSK_Async,TRUE tag,value pair in the taglist, this function is asynchroneous. It will return immediately. If you don't include this tag,value pair, the function is synchroneous. That is, it waits until the main task has returned from the method.

You may not use this function outside subtask code.

Inputs

Parameter Description
taglist Array of TagItem structures with attribute/value pairs

Result

The return value of the target object's AOM_UPDATE method, if used synchroneously.

If used asynchroneously, the result is undefined.
