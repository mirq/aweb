Aaddchild()

Purpose

Perform the AOM_ADDCHILD method on an object.

Synopsis

   result = Aaddchild(parent, child, relationship)
   D0                 A0      A1     D0

   ULONG Aaddchild(struct Aobject *, struct Aobject *, ULONG)

Function

Object oriented support function that invokes the AOM_ADDCHILD method on the specified parent object, to add the specified child object to the relationship.

Inputs

Parameter Description
parent Pointer to the object that is parent in the relation, this is the object on which the method is invoked
child Pointer to the object to add to the relation
relationship ID of the relationship to add the child object in

Result

Specific to the method and the object's class.
