Replytaskmsg()

Purpose

Reply a previously obtained message within a subtask.

Synopsis

   Replytaskmsg(taskmsg)
                A0

   void Replytaskmsg(struct Taskmsg *)

Function

AOTP_TASK support function. It replies a struct Taskmsg previously obtained by the Gettaskmsg() function.

Before replying the message, you may set its result field to an appropriate value.

You may not use this function outside subtask code.

Inputs

Parameter Description
taskmsg Pointer to a struct Taskmsg that you are done with.

See also

Gettaskmsg()
