Checktaskbreak()

Purpose

Check if this subtask has received a message setting AOTSK_Stop to TRUE.

Synopsis

   stop = Checktaskbreak()
   D0

   BOOL Checktaskbreak(void)

Function

AOTP_TASK support function. It checks if the subtask has been stopped by the main task. The main task can do so by setting the AOTP_TASK object's AOTSK_Stop attribute to TRUE.

You may not use this function outside subtask code.

Result

TRUE, if the subtask was stopped by the main task.
FALSE, otherwise.
