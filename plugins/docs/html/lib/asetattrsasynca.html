AsetattrsasyncA()

Purpose

Set attributes of a subtask in an asynchroneous fashion.

Synopsis

   AsetattrsasyncA(task, taglist)
                   A0    A1

   void AsetattrsasyncA(struct Aobject *, struct TagItem *)

   Asetattrsasync(task, ULONG, ...)

   void Asetattrsasync(struct Aobject *, ULONG, ...)

Function

AOTP_TASK support function. It builds an AOM_SET message using the supplied taglist, and sends it asynchroneously to the subtask.

You may include the AOTSK_Replied attribute in the taglist to have a function called when the message is replied by the subtask.

The Asetattrsasync() variant allows you to build the taglist on the stack.

Inputs

Parameter Description
task Pointer to the AOTP_TASK object that owns the subtask.
taglist Array of TagItem structures with attribute/value pairs.
