Freemem()

Purpose

Free a block of memory allocated before within AWeb's memory system.

Synopsis

   Freemem(memoryblock)
           A0

   void Freemem(void *)

Function

This function returns memory to the system that was previously allocated within AWeb's memory system, using one of the Allocmem() or Dupstr() functions.

Inputs

Parameter Description
memoryblock Pointer to a previously allocated block of memory

See also

Allocmem(), Freemem()
