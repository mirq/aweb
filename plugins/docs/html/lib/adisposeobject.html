Adisposeobject()

Purpose

Perform the AOM_DISPOSE method on an object.

Synopsis

   Adisposeobject(object)
                  A0

   void Adisposeobject(struct Aobject *)

Function

Object oriented support function that invokes the AOM_DISPOSE method on the specified object, to delete the object.

Inputs

Parameter Description
object Pointer to the object that should be disposed. When this function returns this pointer has become invalid and must not be used again.
