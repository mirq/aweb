Waittask()

Purpose

Let a subtask wait for incoming messages, and optionally for other signals.

Synopsis

   received = Waittask(signals)
   D0                  A0

   ULONG Waittask(ULONG)

Function

AOTP_TASK support function. This function lets the subtask wait for any of the specified signals, and any incoming messages sent by the main task.

If messages are queued, or any of the specified signals was set prior to calling this function, it will return immediately.

You may not use this function outside subtask code.

Inputs

Parameter Description
signals Bitmask of the signals to wait for, or 0 to wait only for incoming messages from the main task.

Result

Bitmap of the signals that were set, or 0 if there are only incoming messages.
