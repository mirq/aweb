Erasebg()

Purpose

Erase the background with the current background image or background colour.

Synopsis

   Erasebg(cframe, coords, minx, miny, maxx, maxy)
           A0      A1      D0    D1    D2    D3

   void Erasebg(struct Aobject *, struct Coords *, LONG, LONG, LONG, LONG)

Function

This function erases a rectangle of the specified frame, using the current background image or background colour. The specified rectangle is clipped to the visible display area, as defined by the context rendering information (coords).

Inputs

Parameter Description
cframe Pointer to the context frame object in which you want to erase a rectangle. Usually you get the context frame set in the AOBJ_Cframe attribute.
coords Pointer to a valid rendering context information block, struct Coords. You obtain valid context information using the Clipcoords() function.
(minx, miny), (maxx, maxy) Coordinates of the upper left and lower right corner of the rectangle to erase. These coordinates are relative to the frame (that is, they are in the same coordinate system as your object's AOBJ_Left and AOBJ_Top).

See also
