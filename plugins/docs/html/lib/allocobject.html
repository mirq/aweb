Allocobject()

Purpose

Allocate and initialize instance data for a new object.

Synopsis

   object = Allocobject(classid, size, amset)
   D0                   D0       D1    A0

   void *Allocobject(ULONG, ULONG, struct Amset *)

Function

Object oriented support function that allocates memory for a new object instance. The instance data is initialized with zeroes, and the object's class-ID (or object type) is set to the specified value.

Eventually you dispose the instance data by invoking the AOM_DISPOSE method of your superclass object.

Inputs

Parameter Description
classid ID of the true class the new object instance is of.
size Size in bytes of the object instance data, including the superclass object.
amset Pointer to the AOM_NEW message that caused creation of the object. This message might be used for further initialization of the superclass object data.

Results

A pointer to the new, initialized, instance data, or NULL in case of failure.
