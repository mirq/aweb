AnewobjectA()

Purpose

Perform the AOM_NEW method to create an object.

Synopsis

   object = AnewobjectA(classid, taglist)
   D0                   D0       A1

   void *AnewobjectA(ULONG, struct TagItem *)

   object = Anewobject(classid, tag1, ...)

   void *Anewobject(ULONG, ULONG, ...)

Function

Object oriented support function that creates a new object instance. The function invokes the AOM_NEW method for the specified class id (or object type), using the supplied taglist.

The Anewobject() variant allows you to build the taglist on the stack.

Inputs

Parameter Description
classid Class ID, or object type, of the object to create
taglist Array of TagItem structures with attribute/value pairs

Result

An AWeb object, or NULL in case of failure.

You eventually free the object using Adisposeobject().
