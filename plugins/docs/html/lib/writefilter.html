Writefilter()

Purpose

Write data back from within a filter plugin. (V2.0)

Synopsis

   Writefilter(data, length)
               A0    D0

   Writefilter(char *, long)

Function

This function writes a block of data from within a filter type plugin.

This function can only be called from within the plugins Filterplugin() function. It can be called multiple times, even when no input data was available.

Inputs

Parameter Description
data A block of data to be written. The block should not be null-terminated.
length The length of the data block to be written.

Results

After the plugins Filterplugin() function returns, this data will be processed by AWeb.
