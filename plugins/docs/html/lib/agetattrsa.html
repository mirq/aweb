AgetattrsA()

Purpose

Perform the AOM_GET method on an object.

Synopsis

   result = AgetattrsA(object, taglist)
   D0                  A0      A1

   ULONG AgetattrsA(struct Aobject *, struct TagItem *)

   result = Agetattrs(object, tag1, ...)

   ULONG Agetattrs(struct Aobject *, ULONG, ...)

Function

Object oriented support function that invokes the AOM_GET method on the specified object, to get the current value of the object's attributes.

For each entry in the taglist, ti_Tag identifies the attribute, and ti_Data is a pointer to the long variable where you wish the result to be stored.

The Agetattrs() variant allows you to build the taglist on the stack.

Inputs

Parameter Description
object Pointer to object to invoke the method on
taglist Array of TagItem structures with attribute/storage pointer pairs

Results

Specific to the method and the object's class.
