Dupstr()

Purpose

Dupicate a string into a newly allocated memory block.

Synopsis

   copystring = Dupstr(string, length)
   D0                  A0      D0

   UBYTE *Dupstr(UBYTE *, long)

Function

This function allocates a block of memory within AWeb's memory system large enough to hold the string or the specified length. Then the contents of the supplied string is copied into the new memory block.

Eventually you will free the memory using the Freemem() function.

Inputs

Parameter Description
string A nullbyte terminated string, or any other block of data. Passing a NULL pointer is harmless, and will return NULL.
length The length of the string to be copied. The allocated memory block is at least 1 byte larger, and a nullbyte is added after the specified length has been copied.

If this parameter is negative, the length of the nullbyte terminated string is used.

Results

A pointer to a new block of memory, holding a copy if the supplied string data, or NULL in case of failure.

See also

Allocmem(), Freemem()
