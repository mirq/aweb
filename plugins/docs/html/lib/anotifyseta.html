AnotifysetA()

Purpose

Perform the AOM_NOTIFY method on an object using an adhoc AOM_SET message.

Synopsis

   result = AnotifysetA(object, taglist)
   D0                   A0      A1

   ULONG AnotifysetA(struct Aobject *, struct TagItem *)

   result = Anotifyset(object, tag1, ...)

   ULONG Anotifyset(struct Aobject *, ULONG, ...)

Function

Object oriented support function. First it builds an adhoc AOM_SET message using the supplied taglist. Then it invokes the AOM_NOTIFY method on the specified object, using this adhoc message.

The Anotifyset() variant allows you to build the taglist on the stack.

Inputs

Parameter Description
object Pointer to object to invoke the method on
taglist Array of TagItem structures with attribute/value pairs, to be used in the adhoc AOM_SET message.

Result

Specific to the method and the object's class.
