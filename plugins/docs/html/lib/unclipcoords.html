Upclipcoords()

Purpose

Free previously obtained rendering context information.

Synopsis

   Unclipcoords(coords)
                A0

   void Unclipcoords(struct Coords *)

Function

This function releases a rendering context information block, that was previously obtained using the Clipcoords() function.

Inputs

Parameter Description
coords Pointer to a struct Coords that was previously obtained
