Arender()

Purpose

Perform the AOM_RENDER method on an object.

Synopsis

   result = Arender(object, coords, minx, miny, maxx, maxy, flags, text)
   D0               A0      A1      D0    D1    D2    D3    D4     A2

   ULONG Arender(struct Aobject *, struct Coords *, long, long, long, long,
                 ULONG, void *)

Function

Object oriented support function that invokes the AOM_RENDER method on the specified object, to let the object render itself.

Inputs

Parameter Description
object Pointer to object to invoke the method on
coords Pointer to a struct Coords that contain context info, or NULL
minx, miny, maxx, maxy Coordinates relative to the frame to limit the rendering.
If no limitation is required, use 0 for minima and AMRMAX for maxima.
flags Flags that may affect the rendering.
text Internal use only. You must leave this NULL

Results

Specific to the method and the object's class.
