Updatetask()

Purpose

Send an AWeb OO message from a subtask to the main task.

Synopsis

   result = Updatetask(message)
   D0                  A0 

   ULONG Updatetask(struct Amessage *)

Function

AOTP_TASK support function. It sends the supplied message to the main task, invoking the target object's AOM_UPDATE method. A pointer to the message will appear as value of the AOTSK_Message attribute.

This function is always synchroneous, it waits until the main task has returned from the method.

You may not use this function outside subtask code.

Inputs

Parameter Description
message Pointer to a properly initialized struct Amessage

Result

The return value of the target object's AOM_UPDATE method.
