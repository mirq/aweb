Anotify()

Purpose

Perform the AOM_NOTIFY method on an object.

Synopsis

   result = Anotify(object, message)
   D0               A0      A1

   ULONG Anotify(struct Aobject *, struct Amessage *

Function

Object oriented support function that invokes the AOM_NOTIFY method on the specified object, to send a message to this object and to its children.

Inputs

Parameter Description
object Pointer to object to invoke the method on
message Pointer to a struct Amessage to send to the object and its children.

Result

Specific to the method and the object's class.
