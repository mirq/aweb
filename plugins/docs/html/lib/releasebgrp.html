Releasebgrp()

Purpose

Free previously obtained background RastPort.

Synopsis

   Releasebgrp(bgrp)
               A0

   void Releasebgrp(struct RastPort *)

Function

This function releases a RastPort containing a section of the background, that was previously obtained using the Obtainbgrp() function.

Inputs

Parameter Description
bgrp Pointer to the struct RastPort that was previously obtained

Obtainbgrp()
