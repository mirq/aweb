Awebcommand()

Purpose

Execute an AWeb ARexx command. (V2.0)

Synopsis

   returncode = Awebcommand(portnumber, command, buffer, length)
   D0                       D0          A0       A1      D1

   long Awebcommand(long, char *, char *, long)

Function

This function executes an ARexx command and returns the result.

Inputs

Parameter Description
portnumber
command
buffer
length

Results

A pointer to a new block of memory, holding a copy if the supplied string data, or NULL in case of failure.

See also

Allocmem(), Freemem()
