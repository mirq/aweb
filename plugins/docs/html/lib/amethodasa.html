AmethodasA()

Purpose

Perform a method on a coerced object.

Synopsis

   result = AmethodasA(classid, object, message)
   D0                  D0       A0      A1

   ULONG AmethodasA(ULONG, struct Aobject *, struct Amessage *)

   result = Amethod(classid, object, tag1, ...)

   ULONG Amethod(ULONG, struct Aobject *, ULONG, ...)

Function

Object oriented support function that invokes the supplied message on the specified object, as though it were the specified class.

The Amethodas() variant allows you to build the message on the stack.

Inputs

Parameter Description
classid ID of object class to receive the message
object Pointer to an AWeb OO object
message Pointer to method-specific message to send

Result

Specific to the message and the class.
