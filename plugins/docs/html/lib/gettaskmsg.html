Gettaskmsg()

Purpose

Get a queued message within a subtask.

Synopsis

   taskmsg = Gettaskmsg()
   D0

   struct Taskmsg *Gettaskmsg(void)

Function

AOTP_TASK support function. It checks the subtask's internal message port for any queued messages, and return the first one.

When you are done with the message, you must use the Replytaskmsg() function to reply it.

You may not use this function outside subtask code.

Result

A pointer to a struct Taskmsg, or NULL of no message was waiting.

See also

Checktaskbreak(), Replytaskmsg(), Waittask()
