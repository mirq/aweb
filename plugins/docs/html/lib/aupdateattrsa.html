AupdateattrsA()

Purpose

Perform the AOM_UPDATE method on an object.

Synopsis

   result = AupdateattrsA(object, taglist)
   D0                  A0      A1

   ULONG AupdateattrsA(struct Aobject *, struct TagItem *)

   result = Aupdateattrs(object, tag1, ...)

   ULONG Aupdateattrs(struct Aobject *, ULONG, ...)

Function

Object oriented support function that invokes the AOM_UPDATE method on the specified object, to update the object's attributes.

The Aupdateattrs() variant allows you to build the taglist on the stack.

Inputs

Parameter Description
object Pointer to object to invoke the method on
taglist Array of TagItem structures with attribute/value pairs

Results

Specific to the method and the object's class.
