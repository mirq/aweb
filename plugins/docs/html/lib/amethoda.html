AmethodA()

Purpose

Perform a method on an object.

Synopsis

   result = AmethodA(object, message)
   D0                A0      A1

   ULONG AmethodA(struct Aobject *, struct Amessage *)

   result = Amethod(object, tag1, ...)

   ULONG Amethod(struct Aobject *, ULONG, ...)

Function

Object oriented support function that invokes the supplied message on the specified object. The message is invoked on the object's true class.

The Amethod() variant allows you to build the message on the stack.

Inputs

Parameter Description
object Pointer to an AWeb OO object
message Pointer to method-specific message to send

Result

Specific to the message and the object's class.
