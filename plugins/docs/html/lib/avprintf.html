Avprintf()

Purpose

Output a debug message to the standard output stream.

Synopsis

   length = Avprintf(format, arguments)
   D0                A0      A1

   long Avprintf(UBYTE *, va_arg)

   length = Aprintf(format, ...)

   long Aprintf(UBYTE *, ...)

Function

In order to help debugging of plugin modules, this function outputs a formatted string to the standard output stream of AWeb. The Avprintf() function is used exactly like the standard vprintf() function, that cannot be used from within a shared library context.

The Aprintf() variant allows you to pass the arguments on the stack, just like the standard printf() function.

Do not use this function in released plugins!
 It is only there to make the debugging of your code easier.

Obviously this function only works if there is a standard output stream, that is if AWeb was started from a shell. You can use output redirection.

Although not really supported, this function also works from within subtasks. Beware however that the output will mess up if two tasks are using this function simutaneously.

Inputs

Parameter Description
format A standard printf() type formatting string
arguments Array of arguments to the formatting string

Result

The number of bytes written to the standard output stream.
