Awebplugin library

Purpose

The awebsupport.library contains some vital functions for plugin implementations to use.

Library functions

Only the functions labelled "Yes" in the Sub column are allowed within a subtask.

The Ver column shows the minimum version of the library that supports this function.

Using the library

Before any of the functions can be used, the library must be opened: 

   AwebSupportBase = OpenLibrary("awebsupport.library",2)

Note that the library is only available when AWeb is running. Also, any attempt to use the library from within other programs, not AWeb plugins, will most likely cause a crash.

If a plugin has opened the library, it must close it before expunging, or else AWeb will not be able to quit.
