Aremchild()

Purpose

Perform the AOM_REMCHILD method on an object.

Synopsis

   result = Aremchild(parent, child, relationship)
   D0                 A0      A1     D0

   ULONG Aremchild(struct Aobject *, struct Aobject *, ULONG)

Function

Object oriented support function that invokes the AOM_REMCHILD method on the specified parent object, to remove the specified child object from the relationship.

Inputs

Parameter Description
parent Pointer to the object that is parent in the relation, this is the object on which the method is invoked
child Pointer to the object to remove from the relation
relationship ID of the relationship to remove the child object from

Result

Specific to the method and the object's class.
