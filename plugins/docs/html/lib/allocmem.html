Allocmem()

Purpose

Allocate memory within AWeb's memory system.

Synopsis

   memoryblock = Allocmem(size, attributes)
   D0                     D0    D1

   void *Allocmem(ULONG, ULONG)

Function

This function allocates memory within AWeb's memory system. It is used exactly like exec.library's AllocVec() function.

Eventually you will free the memory using the Freemem() function.

Plugin implementations generally need not use this function (although it is allowed to do so), except for cases where it is documented that this function must be used to allocate memory.

Inputs

Parameter Description
size The size of the desired memory block in bytes.
attributes Currently only the MEMF_CHIP flag is recognized to allocate chip memory. If this flag is not set, fast memory is allocated by default, if available.

All memory allocated by this function is public, and is cleared after allocation.

Result

A pointer to a newly allocated block of memory of at least the required size, or NULL in case of failure.

See also

Dupstr(), Freemem()
