Method: AOM_DISPOSE

Purpose

The AOM_DISPOSE method is used to delete an object instance.

Message structure

The AOM_DISPOSE method uses a standard Amessage structure.

Usage

Return value

The return value for this method is not defined.

After sending this message, the object does no longer exist.

Implementation

All classes must implement this method.

Typically, the class frees all resources allocated by the object, then passes this method to its superclass object. The root class AOTP_OBJECT will delete the memory occupied by the object data structure itself.
