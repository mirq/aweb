Method: AOM_NOTIFY

Purpose

The AOM_NOTIFY method is used to send a message to an object, and let the object forward the message to all its children.

Message structure

The AOM_NOTIFY method uses an Amnotify structure:

struct Amnotify
{  struct Amessage amsg;
   struct Amessage *nmsg;
};

Usage

nmsg

This is a pointer to the struct Amessage to be forwarded.

Return value

The return value for this method is not defined.

Implementation

Plugin classes generally need not to implement this method.
