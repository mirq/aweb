Method: AOM_REMCHILD

Purpose

The AOM_REMCHILD method is used to remove a child object from a relationship. The child must have been added to the relation earlier by the AOM_ADDCHILD method.

Message structure

The AOM_REMCHILD method uses an Amadd structure:

struct Amadd
{  struct Amessage amsg;
   struct Aobject *child;
   ULONG relation;
};

Usage

child

The child object to be removed from the relationship.

relation

Identifies the relationship to remove the child object from.

Return value

The return value for this method is not defined.

Implementation

Plugin classes generally need not to implement this method.
