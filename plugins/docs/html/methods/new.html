Method: AOM_NEW

Purpose

The AOM_NEW method is used to create a new instance of a class.

Message structure

The AOM_NEW method uses an Amset structure:

struct Amset
{  struct Amessage amsg;
   struct TagItem *tags;
};

Usage

Message

The AOM_NEW message is sent to a NULL object, and can therefore only be used with the AmethodasA() function. Most convenient is to use the varargs stub function Anewobject().

tags

The taglist holds the tags that set the attributes of the newly created object.

Return value

The method returns a pointer to the new object, or NULL if creation failed.

Implementation

All classes must implement this method.

Typically, the class allocates a new instance using the Allocobject() function, then does some general initialization and calls the code for the AOM_SET method.
