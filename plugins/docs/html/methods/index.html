Methods

Defined object methods

Method Description
AOM_ADDCHILD Add object as a child into a relationship with another object
AOM_DISPOSE Delete an object instance
AOM_GET Get attribute values from an object
AOM_GOACTIVE Make copydriver the input focus
AOM_GOINACTIVE Remove copydriver from the input focus
AOM_HANDLEINPUT Let copydriver handle an input event
AOM_HITTEST Check if copydriver understand input event
AOM_INSTALL Install the dispatcher for a class
AOM_LAYOUT Let copydriver compose itself
AOM_MEASURE Get dimensions from copydriver
AOM_NEW Create a new instance
AOM_NOTIFY Send message to an object and all its children
AOM_REMCHILD Remove object from a relationship with another object
AOM_RENDER Let copydriver render itself
AOM_SET Set attributes of an object
AOM_SRCUPDATE Send data to sourcedriver
AOM_UPDATE Set attributes of a related object

Message structure

Every message used to trigger a method starts with the Amessage structure:

struct Amessage
{  ULONG method;
};
The field method holds the method to identify the type of message.
