Method: AOM_GET

Purpose

The AOM_GET method is used to inquire the value of attributes from an object.

Message structure

The AOM_GET method uses an Amset structure:

struct Amset
{  struct Amessage amsg;
   struct TagItem *tags;
};

Usage

Tags

The taglist holds the requested attribute ID in ti_Tag, and a pointer in ti_Data. The value will be stored in the longword pointed to by this pointer.

Some classes use a <tag,value> pair to set additional parameters that influence following requested attribute values.

Return value

The return value for this method is not defined.

Implementation

All classes should implement this method.

Typically, the class scans the taglist, and stores attribute values in the locations pointed to.
