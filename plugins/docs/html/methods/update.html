Method: AOM_UPDATE

Purpose

The AOM_UPDATE method is used to set attributes of a related object. An example is a scroller that sends a new position to its owner.

Message structure

The AOM_UPDATE method uses an Amset structure:

struct Amset
{  struct Amessage amsg;
   struct TagItem *tags;
};

Usage

tags

The taglist holds the tags that set the attributes of the object.

Return value

The return value for this method is not defined.

Implementation

A class should only implement this method if it uses objects that submit AOM_UPDATE messages.

An AOM_UPDATE message needs not be forwarded to the superclass object.

The class scans the taglist, changes its attributes accordingly, and does any necessary resulting processing.
