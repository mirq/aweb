Method: AOM_SET

Purpose

The AOM_SET method is used to set attributes of an object.

Message structure

The AOM_SET method uses an Amset structure:

struct Amset
{  struct Amessage amsg;
   struct TagItem *tags;
};

Usage

tags

The taglist holds the tags that set the attributes of the object.

Return value

The return value for this method is not defined.

Implementation

All classes should implement this method.

Typically, the class first sends a new AOM_SET message with the taglist from the Amset message to its superclass object. NOTE: Sending the same Amset message will not work correctly if you use the same code to process the tags for the AOM_SET and AOM_NEW methods.

Then the class scans the taglist, changes its attributes accordingly, and does any necessary resulting processing.
