Method: AOM_LAYOUT

Purpose

The AOM_LAYOUT method is used to make the copydriver object layout itself.

Message structure

The AOM_LAYOUT method uses an Amlayout structure:

struct Amlayout
{  struct Amessage amsg;
   LONG width,height;      /* Target width and height to layout within */
   USHORT flags;
   APTR text;
   LONG startx;
   struct Amlresult *amlr;
};

Usage

width, height

These are the target width and height of the current window or frame.

Return value

The return value for this method is not defined.

Implementation

Only copydriver objects must implement this method.

The object should layout itself and compute all its components coordinates relative to the copydriver object. Then set its AOBJ_Width and AOBJ_Height attributes (the struct Copydriver's aow and aoh fields).

Note that the AOBJ_Left and AOBJ_Top attributes will be changed by other methods processed by your superclass object. Never cache coordinates other than relative coordinates to your outer bounds.
