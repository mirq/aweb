Method: AOM_ADDCHILD

Purpose

The AOM_ADDCHILD method is used to add an object (the child) into a relationship with another object (the parent). Relationships are generally of order 1:N, that is one parent can be related to many child objects.

Eventuelly the child must be removed from the relationship by the AOM_REMCHILD method.

Message structure

The AOM_ADDCHILD method uses an Amadd structure:

struct Amadd
{  struct Amessage amsg;
   struct Aobject *child;
   ULONG relation;
};

Usage

Note that the publicly available relationships will not use the object's next and prev fields (the embedded MinNode structure that every object starts with) to implement the relationship. Therefore these fields remain free for your own use.

Field: child

The child object to be added into the relationship.

Field: relation

Identifies the relationship to add the child object into. In general, an object can be parent in more than one relationship.

Return value

The return value for this method is not defined.

Implementation

Plugin classes generally need not to implement this method.
