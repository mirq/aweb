Method: AOM_SRCUPDATE

Purpose

The AOM_SRCUPDATE method is used to send input to a sourcedriver.

Message structure

The AOM_SRCUPDATE method uses an Amsrcupdate structure:

struct Amsrcupdate
{  struct Amessage amsg;
   APTR fetch;
   struct TagItem *tags;
};

Usage

Plugin classes will never send an AOM_SRCUPDATE message.

tags

The taglist holds the tags that set the attributes of the object.

Return value

The return value for this method is not defined.

Implementation

Only sourcedrivers must implement this method.

An AOM_SRCUPDATE message needs not be forwarded to the superclass object.

The class scans the taglist, changes its attributes accordingly, and does any necessary resulting processing.
