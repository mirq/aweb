AOTP_SOURCE class

Purpose

The AOTP_SOURCE class represents the in memory copy of a loaded file. It delegates most of its job to the attached AOTP_SOURCEDRIVER object.

Supported methods

Method Description
AOM_NOTIFY Notify related AOTP_COPY objects
AOM_SET Set attribute values.

Supported attributes

Attribute Type Description Applicability
AOSRC_Memory ULONG Total amount of memory occupied by the attached AOTP_SOURCEDRIVER object. AOM_SET

Usage

Whenever a AOTP_SOURCEDRIVER subclass object changes the amount of memory occupied for the internal representation of the file, it should set the AOSRC_Memory attribute of the AOTP_SOURCE object that it belongs to. The value of this attribute is the current total amount of memory. This is used in the memory cache managing process.

The AOM_NOTIFY method forwards the supplied nmsg message to all related AOTP_COPY objects. If the forwarded message was AOM_SET, these will in turn forward it to their respective AOTP_COPYDRIVER objects.
