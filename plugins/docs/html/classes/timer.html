AOTP_TIMER class

Purpose

The AOTP_TIMER class is used to get the system time, or receive an AOM_UPDATE message after a specified amount of time.

Supported methods

Method Description
AOM_DISPOSE Dispose the object
AOM_GET Get attribute values
AOM_NEW Create a new object
AOM_SET Set attribute values

Supported attributes

Attribute Type Description Applicability
AOBJ_Map struct TagItem * Array of TagItem structures used to map the attributes send by the timer in an AOM_UPDATE message. AOM_NEW
AOM_SET
AOBJ_Target struct Aobject * Object that the AOM_UPDATE messages from the timer will be sent to. AOM_NEW
AOM_SET
Pointer to the AOTP_TIMER object sending this AOM_UPDATE message. Received on AOM_UPDATE
AOTIM_Ready BOOL This attribute is set in the AOM_UPDATE message send when the specified wait period is over. Received on AOM_UPDATE
AOTIM_Seconds
AOTIM_Micros 		LONG Current system time. AOM_GET
AOTIM_Waitseconds
AOTIM_Waitmicros 		LONG The time period to wait before the AOTP_TIMER object sends an AOM_UPDATE message to its target object. AOM_SET
AOM_GET

Usage

Objects can use this class to time their actions. If the AOTIM_Waitseconds or AOTIM_Waitmicros attributes are set, the AOTP_TIMER object will send an AOM_UPDATE message to the object specified in the AOBJ_Target attribute after the specified period.

After receiving this AOM_UPDATE message you may set the AOTIM_Waitseconds or AOTIM_Waitmicros again to start a new wait period.
