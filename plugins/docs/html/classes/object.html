AOTP_OBJECT class

Purpose

The AOTP_OBJECT class is the root class of AWeb's object oriented hierarchy.

Supported methods

Method Description
AOM_DISPOSE Dispose the object

Supported attributes

The AOTP_OBJECT class does not support any attributes itself. It defines a number of attributes that are used by many of the derived classes, however.

Instance data structure

The AOTP_OBJECT class uses the following data structure: 

struct Aobject
{  APTR next;
   APTR prev;
   SHORT objecttype;
};
Every object that is a direct subclass of AOTP_OBJECT must have this structure as the first item in its instance data structure.

Usage

Field: next, prev

These two fields effectively form a MinNode structure. You may use these fields to link objects of your own classes in a list.
