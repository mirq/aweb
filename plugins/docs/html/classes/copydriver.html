AOTP_COPYDRIVER class

Purpose

The AOTP_COPYDRIVER class is the object to derive plugin copydriver classes from.

Supported methods

Method Description
AOM_DISPOSE Dispose the object
AOM_GET Get attribute values
AOM_SET Set attribute values

Supported attributes

Attribute Type Description Applicability
AOBJ_Cframe struct Aobject * The context frame in which the copydriver is displayed. If this is not NULL, you are being displayed. If it is NULL, you are not visible. AOM_SET
AOM_GET
AOBJ_Height LONG Height of the visible object in pixels. AOM_SET
AOM_GET
AOBJ_Left LONG X coordinate of the left edge of the visible object. AOM_SET
AOM_GET
AOBJ_Top LONG Y coordinate of the top edge of the visible object. AOM_SET
AOM_GET
AOBJ_Width LONG Width of the visible object in pixels. AOM_SET
AOM_GET

Instance data structure

The AOTP_COPYDRIVER class uses the following data structure: 

struct Copydriver
{  struct Aobject object;     /* Object header */
   APTR extension;            /* Private extension data */
   LONG aox,aoy;              /* Left,top position */
   LONG aow,aoh;              /* Width, height */
   struct Aobject *cframe;    /* Current context frame */
};
Every object that is a direct subclass of AOTP_COPYDRIVER must have this structure as the first item in its instance data structure.

Usage

Field: object

This is the compulsory embedded AOTP_OBJECT instance data.

Field: aox, aoy

The X and Y xoordinates of the left top corner of the visible object. These fields correspond with the AOBJ_Left and AOBJ_Top attributes, respectively.

Direct subclasses of AOTP_COPYDRIVER may read these fields directly, but not change them. Beware that the AOTP_COPYDRIVER class supports more, private, methods that may change these fields, so do not cache them between your method invokations.

Field: aow, aoh

The widht and height of the visible object. These fields correspond with the AOBJ_Width and AOBJ_Height attributes, respectively.

Direct subclasses of AOTP_COPYDRIVER may both read and set these fields directly.

Field: cframe

A pointer to the context frame in which the object is displayed, directly or indirectly because it is embedded in a HTML document. This field corresponds with the AOBJ_Cframe attribute.

Direct subclasses of AOTP_COPYDRIVER may read this field directly, but not change it.
