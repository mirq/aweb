AOTP_SOURCEDRIVER class

Purpose

The AOTP_SOURCEDRIVER class is the object to derive plugin sourcedriver classes from.

Supported methods

Method Description
AOM_DISPOSE Dispose the object
AOM_GET Get attribute values
AOM_SET Set attribute values
AOM_SRCUPDATE Receive new data

Supported attributes

The AOTP_SOURCEDRIVER class does not support any attributes itself. It defines a number of attributes that are used by the derived classes, however.

Instance data structure

The AOTP_SOURCEDRIVER class uses the following data structure: 

struct Sourcedriver
{  struct Aobject object;     /* Object header */
   APTR extension;            /* Private extension data */
};
Every object that is a direct subclass of AOTP_SOURCEDRIVER must have this structure as the first item in its instance data structure.

Usage

Field: object

This is the compulsory embedded AOTP_OBJECT instance data.
