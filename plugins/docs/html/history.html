Release history

API 2.0 : 10-Apr-1999

This API corresponds with AWeb 3.3.
  • Added a new plugin function: Queryplugin() to query the plugin for its supported features.
  • Added plugin commands feature.
  • Added new type of plugin: filter type plugins.
  • Enhanced the library: added Setfiltertype(), Writefilter() and Awebcommand() functions.

API 1.6 : 4-Aug-1997

This API corresponds with AWeb 0.109 beta.
  • Added the AOCPY_Onimgload and AOCPY_Onimganim attributes.
  • Added guidelines to set these attributes, required in image decoders.
  • Adapted the ilbmcopy.c example source to set these attributes correctly.

API 1.5 : 16-May-1997

This API corresponds with AWeb 3.0 (public version).

No changes, just a version bump.

API 1.4 : 8-Apr-1997

This API corresponds with AWeb 0.94 beta (AWeb-II 3.0/PRERELEASE).
  • Added the AOCDV_Objectparam attribute.
  • Added the struct Objectparam data structure.

API 1.3 : 8-Mar-1997

This API corresponds with AWeb 0.90 beta.
  • Changed the name of the AOBJ_Frame attribute into AOBJ_Cframe to remain consistent with AWeb's internals.
  • Changed the name of the "frame" field in the Copydriver structure into "cframe".
  • Made a few textual changes.

API 1.2 : 11-Feb-1997

This API corresponds with AWeb 0.87 beta.
  • Added the Obtainbgrp() and Releasebgrp() functions to the support library.
  • Added the AOCDV_Bgchanged attribute.

API 1.1 : 23-Jan-1997

This API corresponds with AWeb 0.85 beta.
  • Added the AOTP_TIMER object.
  • Fixed a bug in the example source for ilbmsource.c (line 108) that caused the decoder to loop forever when it was stopped.

API 1.0 : 15-Jan-1997

Initial release.
