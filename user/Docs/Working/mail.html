Sending mail

You can use the built-in mailer to send e-mail messages. This includes full support for mailto: type URLs and forms. Reading mail is not supported.

Setup

Before you can use the built-in mailer, you have to configure it in the Mail/News page of the network settings window.

Required settings

The following fields are required:
  • Your e-mail address and
  • your real name.
  • The SMTP mail host that you are going to use, probably this is called "mail." or "smtp." followed by the name of your provider.

Optional settings

The following fields are optional:
  • A reply-to address if reactions in e-mail should go to a different address.
  • Your signature file, the contents of this file is appended to every mail message you send.

Optionally, you can use an external mailer instead of the built-in one.

Writing a message

Whenever you click a mailto: type URL, or you submit a "mailto-form", the new mail form is displayed. You can change the addressee (although this is not recommended), change the subject, and write your message.

If you entered the mailer to write a personal reply to a newsgroup article, the original article is quoted (included in your article). Please spend a few seconds to cut down the quoted text until it only contains the lines that you are actually responding to.

You can customize the quote header that appears above the quoted text.

You can add extra message headers to your article. To do so, check the extra mail headers included checkbox, and add the extra headers to the start of the message body. You must leave a blank line after the last header line to separate the extra headers from the message body.
