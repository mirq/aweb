Network Status Window

Purpose

The Network Status Window shows all pending network and local file accesses. There are possibilities to cancel selected transfers, or all transfers.

Opening the window

You can open the window in three ways:
  • By clicking the network status button
  • By selecting the Control / Network Status... menu item
  • By using the hotkey A ?

    Contents of the list

    The listview contains a line for each pending or queued transfer. The line consists of the filename retrieved (without hostname and path), and the current status.

    Current status can be one of the following:
    Queued      Image will be loaded when network slots become available
    Started     Transfer process has started
    Looking up  Looking up host name
    Connecting  Making connection
    Waiting     Request sent; waiting for response
    99999/99999 Reading

    Cancel transfers

    You can cancel one or more specific transfers by first selecting the entries in the listview, then click the cancel button at the bottom left. You can select more than one entry by holding the SHIFT key while clicking on the second and later entries. SHIFT-clicking on a selected entry will deselect it.

    You can cancel all transfers by clicking the cancel all button at the bottom right.

    Transfer rate

    At the bottom of the window the current transfer rate is shown, in characters per second. The time needed to make the connection, and to wait for the response from the server, is not included in the calculation. The figure is an average over the last minute, but only since the moment AWeb actually receives data from the network.

    Only network transfers are included in this figure, not loading from the cache or from local files.
