Information window

Purpose

The information window shows miscellaneous details about the current document.

Opening the window

You can open the window by selecting the Project / Info... menu item, or by selecting the Info item from the default frame popup menu.

Contents of the window

The information window shows the url of the current document, and a list with details about this document. The list contains the following sections:

Read from AWeb cache
This line is included if the document was read from the AWeb cache, not directly from the network.

SSL library used
This line contains the ID string of the SSL library used if the document was retrieved over a secure connection.

HTTPS cipher method
This line contains the cipher (encryption) method used if the document was retrieved over a secure connection.

Transfer headers
This section lists all the HTTP transfer headers as they were sent by the server.

If the document was read from the AWeb cache, the header information as it was reconstructed by the cache is shown, not the original HTTP headers.

Meta information
This section contains information from <META> tags in the document. These details are purely informational.

Clickable links
This section contains clickable entries. Each entry corresponds with a <LINK> tag in the document. Clicking an entry will generally load another document.
