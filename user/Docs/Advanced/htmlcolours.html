HTML colour names

The table below lists all HTML colour names that AWeb recognizes.

Notes:

  • The example colours are only visible if you have turned on background colours.
  • Because of the number of colours your screen can accommodate, some colours may look identical to others.
  • All names containing the word "gray" have synonyms spelling "grey", but these are not listed in order to keep the table size down.
  • Spaces are allowed in colour names, so "alice blue" is recognized as "aliceblue".
