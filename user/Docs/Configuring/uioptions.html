GUI settings: Options

Show navigation gadgets

Use this checkbox to select whether the navigation gadgets are visible or not.

Show user buttons

The user button row is only visible if this checkbox is selected.

Borderless gadgets

This renders the navigation gadgets without a border. If you choose another image set for the navigation gadgets, it is advisable to use an image set with different selected and unselected states so it is clear when you click on a gadget.

Navigation gadgets: text/image selection

You can choose how the navigation gadgets are displayed - either text only, image only, or both.

Navigation gadgets: placement

LeftThis places the buttons in a double row to the left of the location field and status bar, with the download indicator to the far right.
RightThe near-traditional AWeb look with the buttons in a double row on the right and the download indicator to the far left ie. the reverse of "Left".
Top LeftThis places the buttons in a single row to the left and above the location gadget, with the download indicator on the far left.
Top RightThis places the buttons in a single row to the right and above the location gadget, with the download indicator still on the far left.
Big LeftThis places a reduced set of 5 buttons (Back/Forward/Stop/Reload/Home) in a single row to the left of the location field and status bar, with the download indicator to the far right. You would need to modify your buttons file with larger than standard images for best effect.
Big RightThe opposite of "Big Left", with big buttons placed in a single row to the right and the download indicator to the far left.

Window Borders

StandardBrowser window rendered with standard borders and gadgets.
GadgetlessAll gadgets are removed from the browser window borders.
BorderlessAll browser window borders (and gadgets) are removed, allowing AWeb to appear like it's running in a "kiosk" mode.

Popup menu activation

These checkboxes determine the ways to open the popup menu.

If you check one or more of the "SHIFT key", "ALT key" and "CTRL key" checkboxes, the popup menu will open if you click on a link, image or frame corner while holding one of these keys.

If you check one or more of the "right mouse button" and "middle mouse button" checkboxed, the popup menu will also open if you click a link, image or frame corner with that mouse button.

If you deselect all four checkboxes, the popup menu is disabled.
