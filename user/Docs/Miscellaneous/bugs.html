AWeb bugs

Although AWeb is thoroughly tested, it is very likely that there are some bugs left. At the time of release the following important bugs were known. For the latest information, check the AWeb Open Source Project

- Accessing certain websites sometimes prompts you to save an unrecognized file. You can just click 'Cancel'. OS4 Depot is an example of this. The workaround is to add a trailing '/' slash to the URI.
