Locale

The following files and drawers are used in localization.
aweb.cd Defines all strings for the AWeb-II main program.
compcd Script to compile aweb.cd. This compilation is also done in the smakefile.
locale.h Generated include file for AWeb-II main program.
awebcfg.cd Defines all strings for the AWebCfg program.
compcfgcd Script to compile awebcfg.cd. This compilation is also done in the smakefile.
cfglocale.h Generated include file for the AWebCfg program.
ct/language/aweb.ct ct/language/awebcfg.ct Translation files for each language. An extra drawer "ct/empty" contains empty ct files.
compcat language Script to compile one set of .ct files into .catalog files for one language.
catalogs Contains translated catalog files.
updatect language After editing aweb.ct or awebcfg.ct, use this script to update the .ct files for one language.

The scripts need both the catcomp and flexcat tools.
